## 需要材料
### 每位學生
* ESP8266 (建議 D1 mini) 或 ESP32 (建議 Dev Kit v1)
* USB 連接線 (TypeA + mini)
* 麵包板cX1
* LED燈X8
* 有源蜂鳴器X1 及 無源蜂鳴器X1
* 溫濕度感測器DHT11X1 或 溫濕度壓力感測器BME280X1
* 杜邦線X若干 (公公 及 公母)
* DHT11 或 BME280
### 教室
* MySQL伺服器X1 (XAMPP)
* IP分享器X1
